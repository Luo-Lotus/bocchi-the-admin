前端：

- _语言_ - **Typescript**
- _UI框架_ - **React**
- _脚手架_ - **UmiJs**
- _UI组件_ - **Ant Design**

后端:

- _数据库操作_ - **Prisma**
- _Web框架_ - **fastify**

# Getting Start

1. `pnpm install` 安装依赖
2. `pnpm -F @monorepo/server run prisma-generate` 生成数据库字段类型
3. `pnpm run client-dev` 启动client dev
4. `pnpm run server-dev` 启动server dev
